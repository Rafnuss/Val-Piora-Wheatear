# Val Piora Wheatear
Trajectories of Northern Wheatears breeding in Val Piora.

## How to cite?

Main peer-reviewed publication:
> Rime, Y., Nussbaumer, R., Briedis, M. et al. Multi-sensor geolocators unveil global and local movements in an Alpine-breeding long-distance migrant. Mov Ecol 11, 19 (2023). https://doi.org/10.1186/s40462-023-00381-6

Code and dat supporting the publication:
> Nussbaumer, R., & Rime, Y. (2022). Val-Piora-Wheatear: Trajectories of Northern Wheatears breeding in Val Piora (0.4). Zenodo. https://doi.org/10.5281/zenodo.7471404. 

Geolocator Data Package
> Nussbaumer, R., & Rime, Y. (2024). Geolocator Data Package: Trajectories of Northern Wheatears breeding in Val Piora (Switzerland) and Val Troncea (Italy) [Data set]. Zenodo. https://doi.org/10.5281/zenodo.14364136
